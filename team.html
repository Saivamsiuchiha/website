📋 How to add team members

  1. Open team.html in any text editor.
  2. Find a <div class="member-card"> block and duplicate it.
  3. Update member-name, member-role, member-bio, and the avatar-initials.
  4. To use a real photo: create a photos/ folder, drop the image in, then replace the <div class="avatar-placeholder"> block with <img src="photos/yourfile.jpg" alt="Name"/>.
  5. Update LinkedIn / Twitter href attributes in member-socials.
  6. Save and refresh in browser — done.
